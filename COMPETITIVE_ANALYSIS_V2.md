## 四、功能對應架構

### 4.1 模型分層配置

```python
# 新增 config.yaml 支援
routing:
  defaults:
    primary: "claude-sonnet"
    fallback:
      - "gpt-4o"
      - "gemini-2.0-flash"
  
  heartbeat:
    model: "gemini-flash-lite"
    budget: "low"
  
  subagent:
    model: "deepseek-r1"
    budget: "low"
  
  task_overrides:
    CODE_REVIEW: "claude-opus"
    TRANSLATION: "minimax-abab"
    CONVERSATION: "gpt-3.5-turbo"
```

### 4.2 自動 Failover

```python
class AutoFailoverRouter:
    def route(self, task, providers):
        for provider in providers:
            try:
                result = self.call(provider, task)
                return result
            except RateLimitError:
                continue  # 自動切下一個
            except APIError as e:
                if e.recoverable:
                    continue
                raise
        raise AllProvidersFailedError("所有 provider 都失敗")
```
